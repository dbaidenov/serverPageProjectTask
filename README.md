# serverPageProjectTask

Проект посвященный для страницы о серверах хостинговой компании. 

Написано на React & Typescript

1. Абсолютно все стили написаны без всяких библиотек голыми руками на scss/sass, дабы показать навыки верстки (кроме стили бесконечной карусели, для него использовал Ant Design)
2. Добавлена возможность конвертации валюты (там лимитированное количество запросов, по моему токену осталось еще примерно 1200 запросов)
3. Добавлена возможность смена аккаунта с использованием таймера
4. Добавлена возможность перевода языка страницы через i18next (перевел лишь некоторые части страницы для наглядности, а не все. Цель - демонстрация возможности перевода)
5. react-intersection-observer для улучшения стилизации и производительности
6. Использован zustand вместо redux
7. Форматирование чисел в соответствии с различными локализациями
8. Логика алерта при превышении числа покупок больше 500
9. Все адаптивно
10. Логика инпутов как на форме входа в gmail
11. Так как не было макета и дизайна, компоненты, выборка правильной цветограммы и тд все придумал сам по фантазии.
